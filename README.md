# 💼 Employee Wage Simulation (Python)

## 📌 Project Overview
This project simulates an Employee Wage Computation System using Python.
It is implemented step-by-step using different Use Cases (UC1–UC6) to demonstrate core programming concepts without using OOP.

---

## 🚀 Features / Use Cases

### 🔹 UC1: Attendance Check
- Checks whether an employee is Present or Absent using random function.

### 🔹 UC2: Daily Wage Calculation
- Calculates daily wage based on:
  - Wage per hour
  - Full day working hours

### 🔹 UC3: Part-Time Employee
- Adds support for:
  - Full-time employee
  - Part-time employee
- Calculates wage accordingly

### 🔹 UC4: Switch Case Logic
- Implements employee type logic using Python dictionary (switch alternative)

### 🔹 UC5: Monthly Wage Calculation
- Calculates total wage for:
  - 20 working days

### 🔹 UC6: Max Hours / Days Condition
- Computes wage based on:
  - Maximum working hours (100)
  - Maximum working days (20)
